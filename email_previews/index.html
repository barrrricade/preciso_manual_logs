Holiday Application Email Templates

About These Templates

These HTML email templates were extracted from the EmailSystem.gs Google Apps Script file. Each template represents a different type of notification sent by the Holiday Application System:

  • Manager emails - Require action (approve/reject)
  • HR emails - For records and monitoring
  • Employee emails - Confirmations and decisions
  • Error emails - System error notifications

All variables from the original Google Apps Script have been replaced with sample data for preview purposes.

Technical Information

Source Files:

  • EmailSystem.gs - Contains all email template functions
  • GlobalConfig.gs - Contains styling constants and configuration

Variables Replaced:

  • Employee names, emails, departments
  • Company name (set to "Acme Corporation")
  • Request IDs, dates, and working days
  • Manager names and decision makers
  • All color and font constants from GlobalConfig.gs
